# Downfall of the Darkest Destructive Doom (CPP)
 DDDDCPP is a port of the DDDD to C++, using libtcod.

## Controls

|         Movement       |    Fullscreen    | Exit |
|:----------------------:|:----------------:|:----:|
| q, w, e, a, d, z, x, c | Left Alt + Enter | ESC |
